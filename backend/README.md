# MindGarden Backend (scaffold)

Minimal Express + TypeScript scaffold for the MindGarden API.

Quick start

1. From project root, change to backend folder:

```powershell
cd backend
```

2. Install dependencies:

```powershell
npm install
```

3. Start in dev mode:

```powershell
npm run dev
```

The server runs on `http://localhost:4000` by default. Open `/health` to check.
