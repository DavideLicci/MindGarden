# Import Wireframes SVG — Figma Plugin

Questo plugin importa SVG (o ZIP contenenti SVG) e crea frame corrispondenti nella pagina Figma.

Come usare
- Installa in Figma (Developer → Link existing plugin) puntando alla cartella `figma-plugin`.
- Avvia il plugin e trascina SVG o ZIP contenenti SVG. Premi `Importa in Figma`.

Notes
- Il plugin include `libs/jszip.min.js` per funzionare offline dopo il bundle.
- Non memorizza o invia dati a server esterni.
